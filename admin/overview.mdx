# Admin Panel Overview

The UCC Control de Acceso admin panel provides comprehensive tools for managing users, reviewing incidents, and generating reports.

## Who Can Access?

The admin panel is restricted to **authorized UCC employees** with administrative permissions.

## Key Metrics Explained

<AccordionGroup>
  <Accordion title="🔴 Bloqueados (Blocked Users)">
    **Definition**: Users with `acceso = 'bloqueado'`

    **Why It Matters**: These users require immediate attention and manual unblocking.

    **SQL Query**:
    ```sql
    SELECT COUNT(*) FROM usuarios WHERE acceso = 'bloqueado';
    ```

    **Action Required**: Review cases, unblock if appropriate.
  </Accordion>

  <Accordion title="🟡 En Riesgo (At Risk)">
    **Definition**: Active users with exactly 3 failures

    **Why It Matters**: One more incident will trigger automatic blocking.

    **SQL Query**:
    ```sql
    SELECT COUNT(*) FROM usuarios 
    WHERE total_fallas = 3 AND acceso = 'activo';
    ```

    **Action Required**: Monitor closely, consider preventive outreach.
  </Accordion>

  <Accordion title="Failure Distribution">
    **Purpose**: Understand overall user behavior patterns.

    **Breakdown**:
    - 0 failures: Responsible users
    - 1 failure: Occasional forgetfulness
    - 2 failures: Moderate concern
    - 3 failures: High risk
    - 4+ failures: Blocked

    **Use Case**: Identify if widespread issues exist (e.g., semester start).
  </Accordion>
</AccordionGroup>

## Common Admin Tasks

<Steps>
  <Step title="Unblock a User">
    1. Search for user by ID or name
    2. Review failure history
    3. Click "Desbloquear Usuario"
    4. Enter justification (mandatory)
    5. Confirm action
    6. User immediately unblocked, counter reset to 0
  </Step>

  <Step title="Generate Monthly Report">
    1. Navigate to "Reportes"
    2. Select report type (e.g., "Fallas Mensuales")
    3. Choose date range
    4. Select format (Excel/CSV/PDF)
    5. Click "Generar Reporte"
    6. Download file
  </Step>

  <Step title="Search User">
    1. Enter ID or name in search box
    2. Click search or press Enter
    3. View user profile with:
       - All roles
       - Failure history
       - Current status
       - Complete information
  </Step>

  <Step title="Start New Semester">
    1. Export current semester data (backup)
    2. Navigate to "Gestión de Semestres"
    3. Click "Nuevo Semestre"
    4. Confirm archive of current data
    5. Upload new semester CSV files
    6. Verify data loaded correctly
  </Step>
</Steps>

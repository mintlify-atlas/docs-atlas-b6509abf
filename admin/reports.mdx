## Monthly Failure Report

### SQL Query

```sql
SELECT 
    u.id_institucional,
    u.nombre_completo,
    r.nombre_rol,
    COUNT(f.id) as total_fallas,
    COUNT(f.id) FILTER (WHERE f.motivo = 'olvido') as olvidos,
    COUNT(f.id) FILTER (WHERE f.motivo = 'perdida') as perdidas,
    MAX(f.fecha_hora) as ultima_falla,
    u.acceso as estado_actual
FROM usuarios u
JOIN fallas f ON u.id_institucional = f.id_institucional
LEFT JOIN usuario_roles ur ON u.id_institucional = ur.id_institucional
LEFT JOIN roles r ON ur.rol_id = r.id
WHERE DATE_TRUNC('month', f.fecha_hora AT TIME ZONE 'America/Bogota') = '2026-03-01'
GROUP BY u.id_institucional, u.nombre_completo, u.acceso, r.nombre_rol
ORDER BY total_fallas DESC;
```

## Blocked Users Report

### Current Blocked Users

```sql
SELECT 
    u.id_institucional,
    u.nombre_completo,
    u.documento_identidad,
    u.total_fallas,
    MAX(f.fecha_hora) as fecha_bloqueo,
    STRING_AGG(r.nombre_rol, ', ') as roles
FROM usuarios u
JOIN fallas f ON u.id_institucional = f.id_institucional
LEFT JOIN usuario_roles ur ON u.id_institucional = ur.id_institucional
LEFT JOIN roles r ON ur.rol_id = r.id
WHERE u.acceso = 'bloqueado'
GROUP BY u.id_institucional, u.nombre_completo, u.documento_identidad, u.total_fallas
ORDER BY fecha_bloqueo DESC;
```

## By Program (Students)

```sql
SELECT 
    ie.programa,
    COUNT(DISTINCT u.id_institucional) as total_estudiantes,
    COUNT(f.id) as total_fallas,
    COUNT(DISTINCT CASE WHEN u.acceso = 'bloqueado' THEN u.id_institucional END) as bloqueados,
    ROUND(AVG(u.total_fallas), 2) as promedio_fallas
FROM info_estudiante ie
JOIN usuarios u ON ie.id_institucional = u.id_institucional
LEFT JOIN fallas f ON u.id_institucional = f.id_institucional
GROUP BY ie.programa
ORDER BY total_fallas DESC;
```

| Derecho | 189 | 98 | 4 | 0.52 |

## Trend Analysis

```sql
SELECT 
    DATE(fecha_hora AT TIME ZONE 'America/Bogota') as fecha,
    COUNT(*) as total_fallas,
    COUNT(*) FILTER (WHERE motivo = 'olvido') as olvidos,
    COUNT(*) FILTER (WHERE motivo = 'perdida') as perdidas
FROM fallas
WHERE fecha_hora >= NOW() - INTERVAL '90 days'
GROUP BY DATE(fecha_hora AT TIME ZONE 'America/Bogota')
ORDER BY fecha;
```
